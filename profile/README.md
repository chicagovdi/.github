# Chicago VDI User Group

Chicago VDI User Group has been connecting end-user computing professionals IRL in the Chicago area since 2015.

Founded under the Citrix User Group Community (CUGC) banner by Nick Prignano and David Sharp, with Chris Bohner and David Zolnacz joining in the early years, the group has run over a dozen meetups bringing together practitioners working in VDI, AVD, Citrix, EUC, and related technologies.

Darrell Dortch joined the leadership team in 2024 when the group relaunched under the World of EUC banner.

We are volunteer organizers for [World of EUC](https://worldofeuc.org), an independent nonprofit that handles event coordination and community operations. We help plan and promote events, then show up and run them.

Check out the [WEUC events list](https://worldofeuc.org/events) for the next event — hope to see you there!
